# CellVIT
Vision transformers for cell painting assays.

## Dataset
We used a subset of the [Broad Institute Cell Painting Dataset](https://cellpainting-gallery.s3.amazonaws.com/index.html#cpg0019-moshkov-deepprofiler/broad/)

## Access to model weights
https://huggingface.co/cellvit/maxvit-large-tf-224
